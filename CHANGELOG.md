# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-12-03

### Added
- Initial release of the book landing page template
- Responsive hero section with 3D book mockup
- "As Featured In" social proof section
- About the book section with benefits list
- Features grid with Lucide icons
- Chapter preview section
- Author bio with credentials
- Testimonials carousel with star ratings
- Three-tier pricing section (Book, Bundle, Premium)
- PDF Workbook upsell integration
- Newsletter signup with bonus incentive
- FAQ accordion section
- Final call-to-action section
- Fully responsive footer with navigation
- Smooth scroll animations
- Mobile navigation menu
- CSS custom properties for easy theming
- Lucide Icons integration

### Technical
- Pure HTML5, CSS3, and vanilla JavaScript
- No external framework dependencies
- Google Fonts integration (Playfair Display, Inter)
- Mobile-first responsive design
- Cross-browser compatibility
